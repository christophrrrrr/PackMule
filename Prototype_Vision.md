# **Prototype Goal (Version 0.1)**

The purpose of this prototype is NOT to build the full game.

The purpose is to answer one question:

"Is stacking absurd physics objects fun?"

## **Success Criteria**

The player can:

* Start a game.  
* See a base platform.  
* Receive a random object.  
* Rotate the object.  
* Place the object.  
* Receive another random object.  
* Continue stacking.

The game detects:

* Tower collapse.  
* Fallen objects.  
* Score.

## **Content**

For the first prototype:

* 1 map  
* 1 base platform  
* 5-10 objects  
* Basic physics  
* Basic UI

No modifiers.  
No wheel.  
No progression.  
No multiplayer.  
No saving.  
No social features.  
No sound.  
No polish.

## **Question To Answer**

After 10 minutes of playing:

"Do I want to place one more object?"
